## Comment utiliser ce projet ?

Commencez par forker le projet.

Ensuite, comme vous en avez l'habitude en classe, importez le projet dans IntelliJ directement à partir du lien Github. 

Attendez ensuite que Gradle télécharge toutes les dépendances. 

Créer une configuration d'exécution (Run configuration) de type Gradle. Dans l'option de commande à lancer, écrivez
les commandes "clean run".

Vous pouvez maintenant lancer votre projet avec Gradle, en cliquant sur la flèche verte. 

La connexion de votre projet ne fonctionnera pas : il faut importer la BDD, vous savez faire.

Il faut aussi copier le fichier .env.example et renommer la copie .env, puis y placer des valeurs correctes pour 
votre BDD.
